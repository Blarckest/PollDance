Bienvenue sur Polldance. Cette application vous permet de donner votre avis sur les cours de l'ISIMA.

Votre nom d'utilisateur se compose des 2 premières lettres de votre prénom suivies de votre nom de famille. Exemple : Jean Dupont -> jedupont.

Fonctionnement de l'application:

  • Vous pouvez ajouter des avis sur les cours. CHAQUE AVIS DOIT ÉVOQUER UNE SEULE IDÉE.
      Exemple: avis voulant être donné: "Le cours est très intéressant et le professeur est très pédagogue." Ce commentaire doit être séparé en deux avis (opinions) distincts:
    • "Le cours est très intéressant."
    • "Le professeur est très pédagogue."
    Ceci permet d'éviter les doublons et de faciliter la lecture des avis.
  • Vous pouvez consulter les avis des autres utilisateurs. Et voter pour les avis qui vous semblent pertinents. note de 0 à 10. 0: pas d'accord, 5: neutre, 10: tout à fait d'accord.
  • Merci de rester courtois et respectueux dans vos avis. Cela n'empeche pas de donner un avis critique ou négatif.
  • On ne peux pas relier un avis à un utilisateur. Les avis sont anonymes. Tout comme les notes.

Si vous voulez contribuer à l'amélioration de l'application, vous pouvez consulter le code source sur Gitlab.
